# ToDo App with Categories

A full-stack ToDo application that allows users to manage tasks using categories (folders), with a default Inbox for uncategorized or orphaned todos.

-	Default Inbox category:
-	All todos without a selected category are placed in Inbox
-	Todos from deleted categories are automatically moved to Inbox
